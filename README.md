## Run Locally

Clone the project

```bash
  git clone https://github.com/fnxsdummy/Team_Fenixs_Django_2025
```

Go to the project directory

```bash
  cd teamfenixs
```

Install dependencies

```bash
  pip install requirements.txt
```

Start the server

```bash
  python manage.py runserver
```
